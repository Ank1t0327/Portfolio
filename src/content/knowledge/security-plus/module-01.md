# Introduction to Security Concepts

This is the first module in the Security+ study guide.

## The CIA Triad

The core of information security is the CIA triad:
- **Confidentiality**: Ensuring data is not accessed by unauthorized parties.
- **Integrity**: Ensuring data is not altered or tampered with.
- **Availability**: Ensuring data and systems are available to authorized users when needed.
